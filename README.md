# Barrel La Fama - App en desarrollo

⚠️ **App en desarrollo** - algunas funcionalidades aún no están completas y la interfaz puede cambiar.

## Descripción
Esta aplicación está diseñada para la **gestión de productos** de un bar/restaurante. Permite:  

- Añadir productos manualmente a la base de datos.  
- Importar productos desde un fichero CSV.  
- Validaciones básicas de campos (nombre, categoría, precio).  

Está desarrollada en **Android Java** usando **Room** como base de datos local.

## Estado actual

- Importación de productos desde CSV en desarrollo ✅
